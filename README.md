# Adaptive Probabilistic Cellular Automata for Parkinson's Disease Detection

This repository contains the implementation for the research paper:

**Adaptive Probabilistic Cellular Automata with Multi-Channel Speech Features for Parkinson's Disease Detection**

## Features
- Multi-channel speech feature extraction (Mel Spectrogram, GFCC, CQT)
- Principal Component Analysis (PCA)
- Adaptive Probabilistic Cellular Automata (APCA)
- Deep Neural Network (DNN)
- Subject-independent evaluation
- Baseline model comparison

## Dataset
Italian Parkinson's Voice and Speech Dataset.

## Requirements

- Python 3.10+
- PyTorch
- NumPy
- Pandas
- Scikit-learn
- Librosa
- Matplotlib
- tqdm

Install dependencies using:

```bash
pip install -r requirements.txt
```

## Running the Code

Run the main script:

```bash
python apca.py
```
